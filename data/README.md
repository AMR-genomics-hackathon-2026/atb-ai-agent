The data in this folder contains:  
```scheme_species_map_combined.csv``` which maps all MLST names from ATB to their full species name.  
```mlst_processed_all_samples.tsv``` which contains MLST data from the latest ATB update (as of March 27, 2026) as an example. In the future, you can add the latest MLST samples from ATB and ask the agent questions regarding that data.
